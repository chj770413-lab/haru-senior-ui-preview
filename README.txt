하루동행 시니어 UI – 최종 미리보기 버전

- index.html : 시니어용 하루동행 홈 화면 (4개 버튼)
- styles.css : 파란 버튼 / 큰 글씨 / 단일 화면 레이아웃
- ui.js      : 버튼 클릭 시 같은 화면 안에서 아래에 내용이 나타나는 구조
               (복약 / 기분 / 건강 상태 / 하루동행 건강 도우미)

사용 방법
1) 이 폴더를 그대로 압축해서 GitHub 저장소(haru-senior-ui-preview 등)에 업로드합니다.
2) Vercel에서 해당 저장소를 연결한 뒤 재배포합니다.
3) 모바일 Safari/Chrome에서 배포 주소를 열어 UI를 확인합니다.
